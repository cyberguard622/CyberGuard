# 🛡️ CyberGuard - School Cybersecurity Awareness Portal

> **Think Before You Click. Stay Smart. Stay Secure.**

CyberGuard is a modern cybersecurity awareness website designed to educate **students, teachers, and parents** about online safety, cyber threats, and responsible digital behavior.

🌐 **Live Demo:** https://cyberguard622.github.io/CyberGuard

---

# 📖 About the Portal

CyberGuard was developed as a cybersecurity awareness portal to promote safe internet usage and provide educational resources on modern cyber threats.

The website aims to:

* 🔐 Promote cybersecurity awareness
* 🛡️ Encourage safe online behavior
* 📚 Educate users about digital threats
* 🚨 Provide cybersecurity news and updates
* 📝 Allow users to report cyber incidents
* 🌍 Create a safer digital community
